Detailed Description

This class implements the part for merging the dictionaries between the uncompressed delta and the compressed order preserving dictionary. This is done using the following simple algorithm. First a new std::set is created to store all distinct values in a sorted order. Now a new dictionary is created based on this set and a mapping table is build that maps the old values from the old dictionary to the new dictionary.
