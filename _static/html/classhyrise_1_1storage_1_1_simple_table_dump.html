Detailed Description

This is the class that allows dumping a table instance in a very simple way directly to the file system without using a third party library or anything else.

The idea behind this method is that, for a given table a directory is created and inside the directory all information is stored. This means that for all attributes a single file is created that persists all valueIds of the attribute. In addition for each dictionary a file is created that stores all dictionary values in a sorted order. As a last part the metadata of the table is stored in two files. The first file contains the layout for the table and the second table contains the size information for the table.
